Aplicación web para la gestión de citas de barbería y spa, desarrollada en Python con Flask.

## Requisitos

- Python 3.x
- Entorno virtual (recomendado)

## Instalación

```bash
python -m venv venv
venv\Scripts\activate  # Windows
# source venv/bin/activate  # Linux/Mac

pip install -r requirements.txt
